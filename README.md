# Image Recommender – Big Data Engineering

Content-based image recommender system for large-scale image datasets (~500k images).

## Features
- Streaming image generator
- Multiple image similarity metrics
- Approximate nearest neighbor search
- Relational database for image metadata
- Unit tests and profiling

## Setup
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
